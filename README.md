# vue-http

This is a vue project to assist in understanding:

- [x] HTTP requests in vue
- [x] Lifecycle hooks
- [x] Template Refs
- [x] Reusability with mixins

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
